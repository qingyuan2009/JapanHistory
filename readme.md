##  日本战国时代

### 织田信长 おだ のぶなが

织田信长(日文：田 信L 假名：おだ のぶなが，1534年6月23号-1582年6月21号）是日本战国时代贵族大名，出身东海道尾张国守护代织田信秀门下，是大儿子。
从小玩世不恭但权谋惊人。在1560年，把上洛大名今川义元杀掉（桶狭间合战)，之后，扶持足利义昭到畿内山城国京都，从此成为天下顶顶有权势的大名。
之后十几年当中，屡次击败反对他的大名，先后杀脱浅井长政、朝仓义景、武田胜赖等大名，到1582年，已经统治日本本州岛个核心地区。他欢喜颠覆传统，讨厌封建制，
对天皇和京都朝廷态度模棱两可，有传言讲他想把天皇废掉，自家当皇帝。不过，他的野心因为明智光秀的叛变灰飞烟灭，他在1582年6月的本能寺之变当中，织田信长死掉，
他的势力也遭到部将瓜分，最后羽柴秀吉胜出，成为新的天下人。

### 丰臣秀吉 とよとみ ひでよし

丰臣秀吉（日语：N臣 秀吉／とよとみ ひでよし；1537年3月17日―1598年9月18日）是日本安土桃山时代公卿、天正末年至庆长初年期间日本的实际统治者。
丰臣秀吉原为贱民出身，然后因侍奉尾张国（今日本爱知县）的大名织田信长，富有才干而逐渐发迹。原本无姓，成为武士之后改姓“木下”，
后将同僚丹羽长秀的羽字和柴田胜家的柴字各取一字改姓“羽柴”。1582年，主君织田信长于本能寺之变中意外殒命，秀吉其后即于山崎之战中击败叛臣明智光秀，
以拥立织田信长嫡长孙三法师织田秀信为口号，而在清洲会议取得织田家主导地位，并于贱岳之战中击败竞争对手柴田胜家和织田信孝，篡夺织田家原先势力据为己有，
并与织田信雄和德川家康爆发小牧长久手之战，进而成为织田信长的实质继承者。

1585年（天正十三年），羽柴秀吉获日本朝廷封为内大臣兼任关白（摄政），1587年（天正十四年）再兼任朝廷最高官位太政大臣的职位，获赐氏姓“丰臣”，成为日本公卿，
并透过不断征伐与收编各方势力，实现日本自15世纪中叶后首次形式上的统一，是为丰臣政权，为当时日本的最高统治者。在他掌权期间，通过太阁检地、
刀狩令等一系列政策强化武士阶层，稳固其统治基础；并建立起大阪城、在日后取代京都成为日本西部最大的城市，并奠定了今日大阪市的基础。晚年他在庶子丰臣秀赖出生后，
诛杀了原来钦定的继任人丰臣秀次，引起了政权内部动荡；并发动了朝鲜之役，在战事末期逝世，被日本朝廷赐封“丰国大明神”。
纵观日本历史上，从一介贱民成为公卿太政大臣的武将仅丰臣秀吉一人，亦使其与同时代的主君织田信长、及创立江户幕府的德川家康并称为“战国三杰”。
